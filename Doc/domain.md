# 穿衣建議邏輯說明
1. 提取天氣資訊：
   取高體感溫度$T_{max}$、最低體感溫度$T_{min}$、體感溫度最大溫差$\Delta T$、平均濕度$H_{avg}$、最大浦風級$W_{max}$

2. 穿衣邏輯判斷：
2.1 基礎層判斷（以 $T_{max}$ 為基準）若 $T_{max} > 26$°C：建議「短袖」為基礎。若 $20$°C $< T_{max} \le 26$°C：建議「薄長袖」或「短袖 + 薄外套」為基礎。若 $T_{max} \le 19$°C：建議「長袖」為基礎。
2.2 溫差修正（以 $\Delta T$ 為關鍵）$\Delta T < 5$°C (溫差小)：建議「單一厚度穿法」。例如全天長袖。$5$°C $\le \Delta T \le 10$°C (有溫差)：建議「內薄外厚」。提示：「早晚偏涼，中午可脫掉外套」。$\Delta T > 10$°C (溫差極大)：建議「洋蔥式穿法」。提示：「內層短袖，外層需中度保暖（如輕羽絨或防風夾克）」。

3. 環境因素修正（風速與濕度）
3.1 風速判斷（採用 $W_{max}$）只要任一時段 $W_{max} \ge$ 4 級：邏輯：即使體感溫度尚可，風力也會造成瞬間寒冷。建議：強制加入 「防風係數」 標籤。提示：「風力較強，外套請選擇防風材質」。
3.2  濕度判斷（採用 $H_{avg}$）若 $H_{avg} > 75\%$：高溫時：強調「吸濕排汗」。低溫時：提醒「濕冷」，建議多穿一件輕量保暖層。
