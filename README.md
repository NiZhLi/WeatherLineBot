# WeatherLineBot
## 功能
1. 傳送天氣資訊及建議
  * 使用者透過 Line Bot 送訊息（例如「今天台北天氣」）
  * 預設傳送給使用者體感溫度或一天氣溫落差(依後續使用回饋調整)
  * 點選按鈕傳送穿衣建議
  * 點選按鈕傳送**當地**今天/明日天氣
  * 可設定時間(推送時間與需求時間[今明日]推送天氣資訊(二期)
  * 後端呼叫天氣 API，取得即時天氣資訊（溫度、氣溫落差、降雨、體感溫度、風速、濕度）
2. 取得使用者地理位置與時間
3. 生成穿衣/生活建議
  * AI 根據天氣、活動資訊生成文字建議 (例如穿衣、是否攜帶雨具、活動建議)
  * 可以互動式詢問，如「明天適合戶外運動嗎」
  * 依據使用者回饋更新上下文
4. 使用者互動流程
  * 初次設定：
    Bot 問一次：
    1. 常在地點？（家/公司/其他）
    2. 室內環境？（冷氣、暖氣、直曬）
    3. 個人體質？（容易冷/正常/耐熱/易流汗等）
  * 直接開始(直接以體感溫度等生成建議)
  * 每次使用
    1. 使用者發訊息：「明天適合戶外運動嗎」、「今天起床穿什麼？」、「今晚洗澡後睡衣怎麼穿？」、「要不要外出加外套？」等
    2. Bot 可提供 快速選擇（室內/室外、特殊活動）：
      * 使用 Quick Reply 按鈕
      * 不必輸入長句，依據使用者常用地點（home/office/其他）快速選擇
